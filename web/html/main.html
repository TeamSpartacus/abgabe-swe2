Infos über diese Anwendung!

Diese Anwendung wurde im Rahmen des AWI-Projekts erstellt. Die Anwendung dient dazu, dass Mitarbeiter des Unternehmens Geräte reservieren können. Die Anwendung wurde mit HTML, CSS und JavaScript erstellt. Die Anwendung ist für die Verwendung auf einem Desktop optimiert.

Die Anwendung besteht aus zwei Hauptseiten. Die erste Seite ist die Übersichtsseite. Auf dieser Seite können Mitarbeiter die Geräte einsehen und reservieren. Die zweite Seite ist die Reservierungsseite. Auf dieser Seite können Mitarbeiter die Geräte reservieren.
